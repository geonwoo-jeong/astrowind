現代人は目を酷使する生活習慣が一般的で、特にスマートフォンの普及により、目の疲れが増しています。そのため、目の栄養補助食品やサプリメントの需要が高まっており、効果的な製品を見つけることが求められています。この記事では、目の健康に役立つサプリメントについて、有名な研究「AREDS」(Age Related Eye Disease Study)をもとに解説します。

## AREDS（Age Related Eye Disease Study）に基づく目の栄養補助食品

アレッズは、目の老化や疾患に役立つ栄養素が明らかになった世界的に有名な研究です。アレッズ1とアレッズ2の2回にわたり、長期間大規模な調査が行われました。

アレッズ研究により、以下の成分が目の健康に効果的であることが確認されました。

<Image src={image4} alt="AREDS成分" />

- ビタミンC
- ビタミンE
- ルテイン
- ゼアキサンチン
- 亜鉛
- 銅
- オメガ3

目の栄養補助食品やサプリメントとして、ルテインが広く知られていますが、実際には目は複雑な器官であり、ルテインだけでは十分な効果が得られません。アレッズ研究では、ルテイン以外にも様々な成分が使用されており、これらを複合的に摂取することが大切です。

## どのような目のサプリメントを摂取すべきか？

目のサプリメントとして、最低限の成分としてAREDSで確認されたものを摂取することがおすすめです。ただし、目に良い成分はAREDSの成分以外にも多く存在します。

- ビタミン
ビタミンA / ビタミンC / ビタミンD / ビタミンE / ビタミンB-6 / ビオチン

- ミネラル
亜鉛 / セレン / 銅

- 抗酸化成分
βカロテン / ルテイン / ゼアキサンチン / ギンコ・ビロバ / ケルセチン / ビルベリー / アイブライト / ヘスペリジン / アスタキサンチン

- その他
オメガ3

これらの成分を含んだサプリメントを作ると、マルチビタミン+オメガ3+抗酸化成分の組み合わせになります。つまり、目のサプリメントは実質的にマルチビタミンの縮小版であり、目の健康や視力向上に驚くべき効果があると宣伝しているが、上記の特定の成分を1つや2つだけ含んでいる目のサプリメントは望ましくありません。

結論として、目のサプリメントを選ぶ前に、マルチビタミン+オメガ3の組み合わせを基本的に摂取することがおすすめです。これが全体的な健康にも効果的です。
